# Spotify Streaming Analysis Dashboard

##  Purpose

The purpose of this project is to analyze Spotify streaming data and 
identify listening patterns, user engagement, top-performing artists, 
and streaming trends to support data-driven decisions.

##  Business Problem

Music streaming platforms need to understand how users interact with 
songs and artists to improve user experience, increase engagement, and
recommend relevant content. This project converts raw streaming data into
meaningful insights that help identify user behavior, peak listening periods, and popular artists.

##  Tools & Technologies

* Power BI
* Microsoft Excel
* Data Visualization
* DAX
* Power Query

##  Features

* KPI Dashboard
* Monthly Streaming Analysis
* Artist Performance Analysis
* Track Performance Analysis
* Listening Time Analysis
* Skip Rate Analysis
* Listening Category Analysis
* Time-Based User Activity
* Weekday Streaming Analysis
* Interactive Visualizations

##  Key Insights

The analysis shows a total of **150K streams**, **319.98K listening minutes**, and 
**5.33K listening hours**, indicating strong user engagement. The average listening 
time per stream is **2.14 minutes**, while the skip rate remains low at **5.18%**, meaning
most users complete the tracks they play. **September** recorded the highest streaming activity, 
whereas **February** had the lowest. **ZZ Top** emerged as the most-streamed artist, while artists 
such as **Coldplay**, **David Bowie**, and **Imagine Dragons** consistently generated high streams 
and listening hours. Around **61%** of listening sessions were short, while nearly **39%** were long 
sessions. Users were most active during the **morning**, and **Friday** recorded the highest listening 
duration. The analysis also revealed a positive relationship between streams and listening hours, with
listening activity increasing significantly after 2017 and peaking around 2020 before gradually declining 
in recent years.
